# Legible Agents

## Setup

macOS only. Python 3.10+.

```
pip install -r requirements.txt
```

Create `.env`:
```
ANTHROPIC_API_KEY=sk-ant-...
```

In Chrome: **View → Developer → Allow JavaScript from Apple Events** ✓

Grant accessibility permissions to Terminal in **System Settings → Privacy & Security → Accessibility**.

## Run

```
python3 computer_use_agent.py
```

ESC to stop.
